DEPLOYMENT OPTIONS (Choose One)
--------------------------------

1. EASIEST - Netlify (Recommended for beginners)
   - Go to netlify.com
   - Sign up (free)
   - Click "Add new site" → "Deploy manually"
   - Drag and drop this entire folder
   - Done! Your site is live with a free domain
   - Upgrade to custom domain later if needed

2. FREE - GitHub Pages
   - Create GitHub account (if needed)
   - Create new repository
   - Upload all files
   - Enable GitHub Pages in Settings
   - Live at yourusername.github.io/valentyna-bukovska

3. TRADITIONAL - Shared Web Hosting
   - Purchase hosting (GoDaddy, Bluehost, etc.)
   - Use FTP client (FileZilla is free)
   - Connect to hosting
   - Upload all files to public_html folder
   - Visit your domain

4. ADVANCED - Vercel
   - Go to vercel.com
   - Sign up with GitHub
   - Import this project
   - Auto-deployed with custom domain option

TESTING LOCALLY (Before Upload)

CUSTOM DOMAIN
