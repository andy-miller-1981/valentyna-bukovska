# Valentyna Bukovska - Psychology Training Website

## Static HTML/CSS Export

This is a complete static website export ready to deploy on any web hosting platform.

### What's Included

- `index.html` - Main website file (single-page application)
- `assets/` - CSS and JavaScript files
- Image files - All professional photos
- No server required - works on any basic web hosting

### Deployment Instructions

#### Option 1: Basic Web Hosting (cPanel, Shared Hosting, etc.)

1. Download all files from this directory
2. Connect to your hosting via FTP
3. Upload all files to your `public_html` folder
4. Visit your domain - the website will be live

#### Option 2: GitHub Pages (Free)

1. Create a GitHub repository named `valentyna-bukovska`
2. Upload all files to the repository
3. Go to Settings → Pages → Select "Deploy from a branch"
4. Choose `main` branch and `/root` folder
5. Your site will be live at `https://yourusername.github.io/valentyna-bukovska`

#### Option 3: Netlify (Free)

1. Go to https://netlify.com
2. Click "Add new site" → "Deploy manually"
3. Drag and drop this folder
4. Your site will be live immediately with a free domain

#### Option 4: Vercel (Free)

1. Go to https://vercel.com
2. Click "Add New..." → "Project"
3. Upload this folder
4. Your site will be live with a free domain
